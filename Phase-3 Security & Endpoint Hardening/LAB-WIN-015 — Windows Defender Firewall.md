# LAB-WIN-015 — Windows Defender Firewall Policy
 
## Objective
 
Configure Windows Defender Firewall using Intune Endpoint Security policies.
 
---
 
## Navigation Path
 
Intune Admin Center  
Endpoint security → Firewall
 
---
 
## Step-by-Step
 
1. Open Intune Admin Center
 
2. Navigate to:
 
Endpoint security → Firewall
 
3. Create firewall policy
 
4. Configure:
 
Enable firewall for Domain profile  
Enable firewall for Private profile  
Enable firewall for Public profile
 
5. Assign policy to:
 
INT-WIN-Enrolled-Devices
