# Phase 3 — Security & Endpoint Hardening
 
This phase focuses on securing Windows endpoints using Microsoft Intune and Endpoint Security policies.
 
The goal is to harden devices against malware, ransomware, and unauthorized access while aligning with Microsoft security best practices.
 
---
 
## Labs Included
 
LAB-WIN-010 — Security Baselines  
LAB-WIN-011 — BitLocker Encryption Policy  
LAB-WIN-012 — Windows Hello for Business  
LAB-WIN-013 — Attack Surface Reduction (ASR)  
LAB-WIN-014 — Microsoft Defender Antivirus  
LAB-WIN-015 — Windows Defender Firewall
 
---
 
## Learning Objectives
 
After completing this phase you will understand:
 
- How security baselines apply recommended Microsoft hardening settings
- How BitLocker protects device data
- How Windows Hello enables passwordless authentication
- How ASR rules prevent ransomware behaviors
- How Defender Antivirus and Firewall protect endpoints
 
---
 
## Enterprise Relevance
 
These policies form the **core Windows endpoint security layer** used in enterprise environments.
 
They protect devices from:
 
- Malware
- Data theft
- Credential attacks
- Unauthorized network access
