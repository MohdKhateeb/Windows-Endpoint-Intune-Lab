# LAB-WIN-010 — Security Baselines (Windows)
 
## Objective
 
Apply Microsoft recommended security settings to Windows devices using Intune Security Baselines.
 
Security baselines provide a predefined set of hardened security configurations.
 
---
 
## Navigation Path
 
Intune Admin Center  
Endpoint security → Security baselines
 
---
 
## Step-by-Step
 
1. Open Microsoft Intune Admin Center
2. Navigate to:
 
Endpoint security → Security baselines
 
3. Select:
 
Microsoft Defender for Endpoint Baseline
 
4. Click **Create Profile**
 
5. Review baseline settings
 
6. Assign policy to:
 
INT-WIN-Test-Devices
 
7. Create policy
