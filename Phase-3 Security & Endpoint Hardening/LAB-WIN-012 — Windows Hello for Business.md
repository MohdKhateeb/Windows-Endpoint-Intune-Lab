# LAB-WIN-012 — Windows Hello for Business
 
## Objective
 
Enable passwordless authentication using Windows Hello for Business.
 
---
 
## Navigation Path
 
Intune Admin Center  
Devices → Windows → Windows Enrollment → Windows Hello for Business
 
---
 
## Step-by-Step
 
1. Open Intune Admin Center
 
2. Navigate to:
 
Devices → Windows → Windows enrollment
 
3. Select **Windows Hello for Business**
 
4. Configure:
 
Require PIN → Enabled  
Allow biometric authentication → Enabled
 
5. Assign to:
 
INT-WIN-Users

## Real-World Usage
 
WHfB enables passwordless authentication using PIN, fingerprint, or facial recognition.
