# LAB-WIN-017 — Microsoft 365 App Deployment
 
## Objective
 
Deploy Microsoft 365 applications to Windows devices using Intune.
 
---
 
## Navigation Path
 
Intune Admin Center  
Apps → Windows → Add
 
---
 
## Step-by-Step
 
1. Open Intune Admin Center
 
2. Navigate to:
 
Apps → Windows → Add
 
3. Select:
 
App type → Microsoft 365 Apps for Windows
 
4. Configure application suite:
 
- Word
- Excel
- PowerPoint
- Outlook
- Teams
 
5. Configure update channel
 
Recommended: Monthly Enterprise Channel
 
6. Assign application to:
 
INT-WIN-Users
