# LAB-WIN-018 — App Protection Policies
 
## Objective
 
Protect corporate data inside applications using Mobile Application Management (MAM) policies.
 
---
 
## Navigation Path
 
Intune Admin Center  
Apps → App Protection Policies
 
---
 
## Step-by-Step
 
1. Open Intune Admin Center
 
2. Navigate to:
 
Apps → App Protection Policies
 
3. Create new policy
 
4. Select platform:
 
Windows
 
5. Configure settings:
 
- Prevent copy/paste to unmanaged apps
- Require PIN for app access
- Encrypt application data
 
6. Assign to:
 
INT-WIN-Users
