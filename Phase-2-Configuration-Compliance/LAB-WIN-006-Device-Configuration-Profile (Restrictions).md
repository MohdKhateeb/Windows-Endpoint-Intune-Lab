## Objective
 
Apply device configuration settings such as password policies and system restrictions using Microsoft Intune.
 
Configuration profiles allow administrators to standardize device behavior across an organization.
 
---
 
## Navigation Path
 
Intune Admin Center  
Devices → Windows → Configuration Profiles
 
---
 
## Step-by-Step
 
1. Open **Microsoft Intune Admin Center**
 
2. Navigate to:
 
Devices → Windows → Configuration Profiles
 
3. Click **Create Profile**
 
4. Configure:
 
Platform  
Windows 10 and later
 
Profile Type  
Templates
 
Template  
Device Restrictions
 
5. Configure settings:
 
- Password complexity
- Control Panel restrictions
- Lock screen settings
 
6. Assign policy to:
 
INT-WIN-Users
 
7. Create the profile.
