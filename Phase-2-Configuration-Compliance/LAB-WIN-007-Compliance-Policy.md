# LAB-WIN-007 – Compliance Policy
 
## Objective
 
Define compliance conditions that determine whether a device is compliant with organizational security requirements.
 
Compliance status can then be used to control access to corporate resources.
 
---
 
## Navigation Path
 
Intune Admin Center  
Devices → Windows → Compliance Policies
 
---
 
## Step-by-Step
 
1. Open **Microsoft Intune Admin Center**
 
2. Navigate to:
 
Devices → Windows → Compliance Policies
 
3. Click **Create Policy**
 
4. Configure settings:
 
- Require password
- Minimum OS version
- Device threat level
 
5. Configure non-compliance actions:
 
- Mark device non-compliant
- Set remediation grace period
 
6. Assign policy to:
 
INT-WIN-Enrolled-Devices

Compliance policies are used to evaluate device security posture before allowing access to corporate services.
