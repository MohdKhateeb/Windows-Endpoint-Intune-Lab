# LAB-WIN-009 – Settings Catalog Configuration
 
## Objective
 
Configure advanced Windows settings using the Intune Settings Catalog.
 
Settings Catalog provides granular device configuration capabilities.
 
---
 
## Navigation Path
 
Intune Admin Center  
Devices → Windows → Configuration Profiles → Settings Catalog
 
---
 
## Step-by-Step
 
1. Open **Intune Admin Center**
 
2. Navigate to:
 
Devices → Windows → Configuration Profiles
 
3. Click **Create Profile**
 
4. Configure:
 
Platform  
Windows 10 and later
 
Profile Type  
Settings Catalog
 
5. Add settings such as:
 
- Power settings
- Windows Update configuration
- Security restrictions
 
6. Assign to:
 
INT-WIN-Test-Devices
