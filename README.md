# Windows Endpoint Management Lab (Zero → Hero)

Enterprise-style Microsoft Intune and Windows Endpoint Management lab environment.

This repository documents a **hands-on learning path for modern device management** using Microsoft Intune, Microsoft Entra ID, and Windows 10/11.

The labs follow an **enterprise deployment lifecycle** used by real organizations.

---

# Lab Phases

## Phase 0 – Windows Foundation
Tenant validation and group architecture.

- Tenant and licensing validation
- Core security groups for device and user targeting

## Phase 1 – Windows Enrollment
Device enrollment using Intune.

- Automatic MDM Enrollment
- Enrollment restrictions
- Manual enrollment via Company Portal

## Phase 2 – Configuration & Compliance
Device configuration and compliance enforcement.

- Configuration profiles
- Compliance policies
- Conditional Access integration

## Phase 3 – Security Hardening
Endpoint security configuration.

- Security baselines
- BitLocker encryption
- Windows Hello for Business
- Defender Antivirus and Firewall

## Phase 4 – Update Management
Windows servicing model.

- Update Rings
- Feature Update policies

## Phase 5 – Automation & Modern Deployment
Advanced endpoint management.

- Proactive Remediation
- Windows LAPS
- Windows Autopilot
- Enrollment Status Page
- Co-management

---

# Purpose of This Lab

This repository is built to:

- Practice **real enterprise Intune deployments**
- Prepare for **Microsoft MD-102**
- Build a **technical portfolio for Endpoint Engineer roles**
- Document **troubleshooting and architecture decisions**

---

# Architecture Principles

- Users authenticate
- Devices comply
- Policies target devices
- Access targets users
