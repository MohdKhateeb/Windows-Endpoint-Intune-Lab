# Phase 1 – Windows Enrollment
 
This phase focuses on enrolling Windows devices into Microsoft Intune.
 
Device enrollment is the process of registering corporate or personal Windows devices so that they can be managed by Microsoft Intune.
 
This phase covers the three primary enrollment methods used in enterprises.
 
---
 
## Labs Included
 
LAB-WIN-003 – Enable Automatic MDM Enrollment  
LAB-WIN-004 – Configure Windows Enrollment Restrictions  
LAB-WIN-005 – Manual Windows Enrollment using Company Portal
 
---
 
## Learning Objectives
 
After completing this phase you will understand:
 
- How automatic MDM enrollment works
- How device enrollment restrictions control device onboarding
- How to manually enroll existing Windows devices
- Common causes of enrollment failures
 
---
 
## Enterprise Relevance
 
These configurations are mandatory before deploying:
 
- Windows Autopilot
- Compliance policies
- Device configuration profiles
- Conditional Access policies
 
Enrollment is the **foundation of modern endpoint management**.
