# Phase 4 — Application Deployment & Management
 
This phase focuses on deploying and managing applications using Microsoft Intune.
 
Application deployment is a core responsibility for endpoint administrators in enterprise environments.
 
---
 
## Labs Included
 
LAB-WIN-016 — Win32 Application Deployment  
LAB-WIN-017 — Microsoft 365 Application Deployment  
LAB-WIN-018 — App Protection Policies (MAM)  
LAB-WIN-019 — Application Update Management
 
---
 
## Learning Objectives
 
After completing this phase you will understand:
 
- How Win32 applications are packaged and deployed
- How Microsoft 365 apps are managed through Intune
- How Mobile Application Management (MAM) protects corporate data
- How application updates are controlled in enterprise environments
 
---
 
## Enterprise Relevance
 
Application management ensures users receive required tools securely while maintaining organizational security policies.
 
Administrators commonly use Intune to deploy:
 
- Enterprise applications
- Microsoft 365 apps
- Security-controlled applications
- Business-critical software
