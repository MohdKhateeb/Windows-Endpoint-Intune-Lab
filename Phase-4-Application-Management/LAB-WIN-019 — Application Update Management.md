# LAB-WIN-019 — Application Update Management
 
## Objective
 
Manage application updates to ensure devices receive the latest versions.
 
---
 
## Navigation Path
 
Intune Admin Center  
Apps → Monitor → App Install Status
 
---
 
## Step-by-Step
 
1. Open Intune Admin Center
 
2. Navigate to:
 
Apps → Monitor
 
3. Review application deployment status
 
4. Configure update settings for applications
 
5. Monitor installation status across devices
